# Free Asset Sources (for future upgrades)

- Kenney (CC0): https://kenney.nl/assets
- OpenGameArt (filter by CC0/Public Domain): https://opengameart.org/
- itch.io free game assets: https://itch.io/game-assets/free
- Freesound (check license per file): https://freesound.org/
- Pixabay SFX/music: https://pixabay.com/sound-effects/

When replacing placeholders, keep a manifest with source URL, creator name, and license for each asset.

Current in-use third-party assets are documented in `docs/asset-license-manifest.md`.
